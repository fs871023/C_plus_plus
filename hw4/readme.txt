polynomial.cpp

建構子：我先讓head是個什麼都沒有的node，然後讓他指向第一個項，同時建立第一個項，
第一個項的係數和次方是陣列的第一個值。
之後用for來把儲存係數和次方的陣列裡的值(從第二個開始)，用add函式加進多項式內。
最後印出提醒訊息

解構子：我讓current從head開始，跑過每一個node，
然後要刪除的時候，用tempptr這個指標指向我要現在的點
然後讓current 先前進一個node，以免失去所在位置，再將node刪除
最後把current所指到的最後一個刪除

add：我使用了pre 和 current兩個指標來進行，各自代表前一個和現在的位置。
先讓pre=head、current=第一個項
因為要將多項式降冪排列，所以當我們沒有跑到最後面，而且要放進來的項比現在的項小，
我們就繼續往下走
當while停下來時有三種狀況：
1.到多項式的最後面了，代表現在的次方式最小的，
那我們就把現在的值建立一個node放在最後面
2.多項式沒有跑完，但發現遇到同樣的次方，這時候就把係數相加即可
3.多項式沒有跑完，但發現要放進去的次方介於這一個和上一個間，那就建立一個node插入
最後檢查運算後的項係數是否為0，若是，這可以代表這個項實際上是不存在的，
這時候就把這個node刪除

微分：一樣使用pre 和 current兩個指標來進行，各自代表前一個和現在的位置。
使用探索每個node的方式做微分，做完繼續前進做下個node
但是會先看這個項是不是常數(次方=0)，如果是的話就直接刪掉，並小心不要失去位置

印出多項式：用current探索每個node，再把值印出來
e.g. 5x^3 + 1x^1 + 2x^0
